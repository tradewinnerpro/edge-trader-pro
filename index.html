Símbolo TradingView: use o formato EXCHANGE:TICKER — ex: OANDA:XAUUSD
Nome / Apelido Símbolo TradingView Fator Padrão
💡 Como encontrar o símbolo: Acesse tradingview.com, pesquise o ativo e copie o símbolo completo no formato EXCHANGE:TICKER.
💡 Fator Multiplicador: No MetaTrader, botão direito no ativo → Propriedades → Contrato → veja "Valor do Tick" ou "Tamanho do Contrato". Divide o tamanho do contrato por 100 para ter o fator por pip/ponto.
Exemplos: GOLD=10, NASDAQ=20, S&P500=50, DOW JONES=5, PRATA=50.

ALTER TABLE trades ADD COLUMN IF NOT EXISTS tipo_saida TEXT DEFAULT 'pending', ADD COLUMN IF NOT EXISTS resultado_real NUMERIC, ADD COLUMN IF NOT EXISTS regras_quebradas TEXT[] DEFAULT '{}', ADD COLUMN IF NOT EXISTS score INTEGER DEFAULT 1, ADD COLUMN IF NOT EXISTS sessao TEXT DEFAULT 'Nova York', ADD COLUMN IF NOT EXISTS conta_tipo TEXT DEFAULT 'demo';
